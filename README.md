## 今後やりたいこと

今は特に、教育・組織・地方とITの掛け合わせに関心があります。

### IT × 〇〇 × 伴走

ITを「解決策そのもの」ではなく、  
**一緒に前に進むための手段**として考えています。

業界や領域を限定せず、  
課題のある現場に入り込み、  
考え、試し、改善しながら伴走する。

一方的に支援するのではなく、  
同じ目線で悩み、ときには競走相手として刺激し合いながら、  
関わる人や組織が前に進むプロセスに寄り添っていきたいです。
